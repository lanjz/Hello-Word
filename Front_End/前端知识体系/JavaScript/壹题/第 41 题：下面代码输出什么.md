```
var a = 10;
(function () {
    console.log(a) // undefine
    a = 5
    console.log(window.a) // 10
    var a = 20;
    console.log(a) // 20
})()
```

这个要注意自执行函数的这行代码`var a = 20`

因为这里有声明一个`a`，导致这个变量提升，所在当前块级作用域中且在`var a=20`之前访问的`a`将为`undefine`

如果没有`var a = 20`，那么将访问全局的变量`window.a`

```
var a = 10;
(function () {
    console.log(a) // 10
    a = 5
    console.log(window.a) // 5
    a = 20;
    console.log(a) // 20
})()
```
