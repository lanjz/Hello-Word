> [大佬，JavaScript 柯里化，了解一下?](https://juejin.im/post/5af13664f265da0ba266efcf)

```
	function add (x, y) {
		return (x*1 + y*1)
	}
    function currying(fn, ...arg1) {
		if(arg1.length >= fn.length){
			return fn(...arg1)
        }
    	return (...arg2) => {
    		return currying(fn, ...arg1, ...arg2)
        }
    }
	console.log(currying(add, 1)(2,3)) // 3
```
