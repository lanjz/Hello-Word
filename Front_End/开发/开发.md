# 无需Webpack直接使用`import`

## 条件

- Chrome

- 使用`<script type="module">`

  ```javascript
  <script type="module">
      import { test } from './test/test1.js'
      test()
  </script>
  ```

- 必须在服务器环境下才运行页面

  全局安装模块`http-server`:`npm install http-server -g`

## 启动

`http-server -c-1`（只输入http-server的话，更新了代码后，页面不会同步更新）
