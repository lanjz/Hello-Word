# fin-img

### 说明

图片渲染组件

### 属性

| 参数            | 类型     | 说明               | 默认值  |
|---------------|--------|------------------|------|
| width         | string | 宽度(需要带单位 .eg:px) | auto |
| height        | string | 调试(需要带单位 .eg:px) | auto |
| alt           | string | 图片描述             | —    |
| border-radius | string | 圆角               | —    |
| fit           | string | 填充模式             | fill |
