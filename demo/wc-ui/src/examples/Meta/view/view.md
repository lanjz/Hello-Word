# Fin-View 组件

### 说明

View 同 div

### 属性

| 参数     | 说明                 | 类型            | 默认值  |
|--------|--------------------|---------------|------|
| sty  | 对象形式的样式设置          | string、object |   |

### todo

当sty是字符串时，将直接当做内联样式直接作用于 view 标签  
当sty是对象时，由于组件无法直接监控属性变化，因此无法触发渲染重新渲染，因此外层的 sty 的属性变化时请构建一个新对象
