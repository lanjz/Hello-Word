# Layer

容器组件

### 说明

- 作为承载容器组件的元素

- 可灵活设置挂载节点

### Attributes
| 参数             | 说明                     | 类型      | 默认值  |
|---------------- |------------------------ |--------- |--------|
| width           | 容器宽度(需要带单位 .eg:px) | string   | auto   |
| height          | 容器调试(需要带单位 .eg:px) | string   | auto   |
| append-top      | 挂载父节点(body、元素ID)   | string   | 所在节点  |
| display         | display[css]            | string   |    —   |
| position        | position[css]           | string   |    —   |
| left            | 左偏移量                  | string   |    —   |
| top             | 顶部偏移量                | string   |    —   |
| right           | 右偏移量                  | string   |    —   |
| bottom          | 底部偏移量                | string   |    —   |
| z-index         | 层叠顺序                 | number   |    —   |
