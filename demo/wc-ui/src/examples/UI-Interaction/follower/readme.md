# Follower

用于 `select`、`date-picker` 这类组件的弹出层使用，自动定位到父级节点的底部

### Attributes
| 参数            | 说明                     | 类型                      | 默认值           |
|---------------|------------------------|-------------------------|---------------|
| previous       | 如果为 true ,跟随当前节点的下一个节点 | Boolean     | false 跟随上一个节点 |
| next       | 如果为 true ,跟随当前节点的上一个节点 | Boolean |        false 跟随上一个节点       |
